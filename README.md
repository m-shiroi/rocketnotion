# rocketnotion
Service for Rocketbook-Notion integration
## Setup to test notebook
* create new virtualenv
```bash
   $ virtualenv .venv
```

* activate virtualenv
```bash
   $ source .venv/bin/activate
```

* load dependencies
```bash
   $ pip install -r requirements.txt
```

* unizp MNIST dataset
```bash
   $ cd data
   $ unzip *
   $ cd ..
```

* lauch notebook
```bash
   jupyter-notebook
```
